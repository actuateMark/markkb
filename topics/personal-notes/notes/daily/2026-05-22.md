### Tool/process discoveries this session

- **`gh pr edit --body` silently fails** when GitHub's GraphQL response includes the Projects-classic deprecation warning. Symptom: command "succeeds" (returns the deprecation message, no error) but the body never actually updates. Workaround: `gh api --method PATCH repos/<owner>/<repo>/pulls/N -F body=@file.md` — direct REST bypasses the GraphQL path.
- **NR `container_image` tag breakdown via `capture()` regex** is the canonical way to see fleet image-rotation state. Pattern: `SELECT uniqueCount(pod_name) FROM Log WHERE message LIKE '%starting patrol%' SINCE Nh ago FACET capture(container_image, r'.*:(?P<tag>[^:]+)$')`. Surfaces tag-by-tag pod counts for any AP signal in one query.
- **vms-connector local `stage` branch had stale 9-commit divergence from origin/stage** when attempting `git pull`. All local-only commits were already represented on origin via past stage→rearch squash merges. Hard-reset to `origin/stage` was safe. Worth checking for any long-running local clone where stage hasn't been synced in a while.
