# alert-ui

**Repository:** `aegissystems/alert-ui`
**Version:** 1.0.0
**Default branch:** `main`

## Purpose

Vue 3 single-page application for viewing and managing security alerts generated by the Actuate platform. It provides a frame-by-frame alert viewer that displays detection results (people, vehicles, weapons, environmental hazards) from camera feeds processed by [[actuate-ailink|AILink]] and the broader Actuate AI pipeline. The app communicates with backend APIs via Axios, with the base URL injected at build time through `VITE_API_BASE_URL`.

## Tech Stack

- **Framework:** Vue 3 (Composition API)
- **Build tool:** Vite 6.3
- **HTTP client:** Axios
- **Unit testing:** Vitest 3.1 with V8 coverage
- **E2E testing:** Playwright 1.52
- **Linting:** ESLint 9 with `eslint-plugin-vue`
- **Formatting:** Prettier
- **Code quality:** SonarQube (coverage upload + auto-remediation)

| Path | Role |

## Detection Labels

The app defines client-side label categories in `config.js`:

- **Vehicles:** bicycle, truck, bus, car, UPS, USPS, FedEx, school bus, Amazon, DHL, fire truck
- **Weapons:** gun, pistol, knife
- **People:** intruder, loiterer, crowd, fall detection, line crossing
- **Environmental:** fire, smoke, left object, no hard hat

## Deployment

Automated via GitHub Actions (`main.yml`):

1. **Test job:** Installs deps, runs ESLint + Vitest with coverage (continue-on-error)
2. **Deploy job (matrix):** Builds with environment-specific `VITE_API_BASE_URL`, assumes AWS IAM role, syncs `dist/` to S3 bucket, sends Slack notification
3. **Sonar job:** Uploads coverage to SonarQube

Deploys to **two environments** via matrix strategy, each with its own API URL and S3 bucket. PRs run tests and lint only.

## Relationship to Other Services

- **Backend:** Connects to the Actuate API (likely [[actuate-monitoring-api|Monitoring API]] or [[actuate-external-api-repo|External API]]) via the configured `VITE_API_BASE_URL`
- **Data source:** Displays alerts and frames produced by [[actuate-ailink|AILink]] and processed through the Actuate AI pipeline
- **White-label:** Supports Envera as a branded variant with its own entry point
- **Hosting:** Static SPA deployed to AWS S3
