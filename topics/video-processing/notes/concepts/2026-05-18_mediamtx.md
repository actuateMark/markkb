# MediaMTX: real-time media server and proxy

MediaMTX is a zero-dependency real-time media server and media proxy that publishes, reads, proxies, records, and plays back video/audio streams. Conceived as a "media router" between protocol endpoints, with a focus on efficiency and portability. Single Go binary, no interpreter, runs on Linux/Windows/macOS.

## Protocol surface

- **Publish:** [[rtmp-and-srt|SRT]], [[webrtc-deep-dive|WebRTC]], [[rtsp-deep-dive|RTSP]], [[rtmp-and-srt|RTMP]], [[hls-and-dash|HLS]], [[mpeg-ts-over-udp|MPEG-TS]], RTP
- **Read:** [[rtmp-and-srt|SRT]], [[webrtc-deep-dive|WebRTC]], [[rtsp-deep-dive|RTSP]], [[rtmp-and-srt|RTMP]], [[hls-and-dash|HLS]]
- **Automatic conversion** between protocols on the same path
- Compatible publishers: [[ffmpeg-entity|FFmpeg]], [[gstreamer-entity|GStreamer]], OBS, Python, Go, Unity, browsers, Raspberry Pi cameras
- Compatible readers: [[ffmpeg-entity|FFmpeg]], [[gstreamer-entity|GStreamer]], VLC, OBS, Python, Unity, browsers

## Operational features

- Multiple concurrent streams on separate paths
- Hot-reload configuration without disconnecting existing clients
- Always-available streams even when the publisher is offline (placeholder)
- Recording to disk in fMP4 or [[mpeg-ts-over-udp|MPEG-TS]]
- Playback of recorded streams
- Authentication: internal, HTTP, or JWT
- Stream forwarding + request proxying to other servers
- Control API for runtime management
- Prometheus-compatible metrics export
- Per-event hooks (external commands on connect/disconnect/read/publish)

## Why it matters for Actuate

Most-relevant external project surveyed in the 2026-05-11 batch. Sits exactly at the [[rtsp-deep-dive|RTSP]]↔[[webrtc-deep-dive|WebRTC]] translation boundary that any fleet-architecture frame-transport design has to solve. The hot-reload + per-path config model + Prometheus surface make it a viable PoC substrate for replacing or augmenting parts of the connector's puller chain when [[webrtc-deep-dive|WebRTC]] ingest enters scope. Sister tool: [[2026-05-18_go2rtc]] (smaller scope, HA-focused).
